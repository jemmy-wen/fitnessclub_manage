# Changelog

## 2026-06-08 LINE Demo 改版

### LINE 對話（學員端）

- 改成 AI 語意分析，不用再打特定關鍵字，說人話就能觸發
- 請假改為多步驟對話：AI 幫你縮小課程範圍 → 多堂課時出現格狀選單讓你點選 → 確認後自動通知教練
- 查課表、查堂數的回覆下方加上 LIFF 快捷連結
- 堂數查詢加上「加購課程」按鈕
- 聯絡教練有三種方式可選：留言、撥電話、加 LINE
- 右側說明欄改成可點擊的範例語句，展示各種說法都能觸發

### LINE 對話（教練端）

- 查詢待確認請假時，每筆請假直接帶「確認收到」和「拒絕」按鈕，不用再跑去後台
- 確認後學員會自動收到通知
- 教練也有快速選單，跟學員端分開，項目完全不同

### 快速選單（兩端通用）

- 點 + 展開六宮格選單，取代原本的純文字輸入
- 需要開介面的（預約、個人資料、請假）直接跳 LIFF，不走對話
- 純查詢的（堂數、聯絡教練）才走對話

### Demo 資料

- 王小明多加了一堂週六課，讓「我這週想請假」可以示範多課程選擇的格狀選單效果
